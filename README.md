# pr-assets — branche d'assets pour les descriptions de PR

Cette branche **orpheline** (aucun lien avec `main`, jamais mergée) héberge
les captures d'écran référencées dans les descriptions de pull requests.

- Un dossier par PR : `pr/<numéro>/<nom>.png`
- URL à utiliser dans la description :
  `https://raw.githubusercontent.com/isc/bemol-osr/pr-assets/pr/<numéro>/<nom>.png`
- On n'y supprime rien (les descriptions de PR mergées continuent d'y faire
  référence).

Voir la section « Captures d'écran dans les PRs » de `CLAUDE.md` (branche
`main`) pour la marche à suivre complète.
